## Portfolio Website
Built with Next.js and Tailwind CSS. It showcases a little about me and a few projects I have been working on.

# Technologies Used
• Next.js
• React.js for building components
• Tailwind CSS for styling

# About
app/page.js: This file represents the homepage of my website. Included projects will be displayed using React components. At the moment, I have placeholder text and a solid image background until I populate the component with the proper data.

(A dynamic routing system will be used to represent individual projects)

Components file: Contains all the components that make up my website. 

app/layout.js: This file provides a basic layout structure for the site, at the moment my website is only one page, so I did not wrap the children(pages) with a header and footer to populate every page.
